# 项目概述 (Project Overview)

## 项目简介

**X-Panel** 是一个基于 Go 语言开发的高性能代理管理面板，提供 Web 可视化界面来管理 Xray 核心的多协议代理服务。

本项目是 [MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) 的优化分支版本，专注于提升用户体验和功能稳定性。

## 核心目标

1. **易用性**: 提供直观的 Web UI，降低代理服务配置门槛
2. **多协议支持**: 全面支持 Xray 核心的各类协议
3. **安全可靠**: 内置完善的安全机制和备份功能
4. **可扩展性**: 模块化架构，便于功能扩展和定制

## 技术栈

### 后端 (Backend)
| 技术 | 用途 |
|------|------|
| **Go 1.21+** | 主要开发语言 |
| **Gin** | Web 框架 |
| **GORM** | ORM 框架，数据库操作 |
| **SQLite** | 默认数据库存储 |
| **Xray-core** | 代理核心引擎 |

### 前端 (Frontend)
| 技术 | 用途 |
|------|------|
| **HTML5/CSS3** | 页面结构与样式 |
| **JavaScript** | 交互逻辑 |
| **Biome** | 代码格式化工具 |

### 运维 (DevOps)
| 技术 | 用途 |
|------|------|
| **Docker** | 容器化部署 |
| **Bash** | 安装/管理脚本 (`install.sh`, `x-ui.sh`) |
| **systemd** | 服务管理 |
| **GitHub Actions** | CI/CD 流水线 |

## 核心功能

- **多协议管理**: VMess, VLESS, Trojan, Shadowsocks, WireGuard 等
- **XTLS/REALITY**: 原生支持先进的传输层安全技术
- **流量统计**: 实时/历史流量监控，支持自动重置
- **客户端限制**: IP/设备数量限制，独立限速
- **配置生成**: 一键生成客户端节点配置
- **Telegram 机器人**: 通知、备份、远程管理
- **订阅转换**: 支持 Clash/Surge 等订阅格式
- **证书管理**: 自动申请 Let's Encrypt 域名/IP 证书
- **多语言支持**: 中文、英文、波斯语等 11 种语言

## 系统要求

- **操作系统**: Ubuntu 20.04+, Debian 11+, CentOS 8+
- **架构支持**: amd64, arm64, armv7
- **最低配置**: 1 核 CPU, 1GB 内存
