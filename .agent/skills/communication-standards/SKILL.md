# 1. 沟通规范 (Communication Standards)

- 所有向用户提交的汇报、说明、文档、回复和最终输出**必须使用中文**。
- 任务执行过程中，必须通过 `task_boundary` 和 `notify_user` 提供及时的中文进度反馈。

---

# 7. 其他强制要求

- 所有输出、计划、Artifacts 必须使用中文。
- 违反以上任何规则，将导致代理任务立即终止，并向用户报告违规原因。
